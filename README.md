# NetPractice

## Description

NetPractice is a project from the 42 curriculum where the goal is to complete 10 levels of network configuration exercises. Each level contains a networking problem that must be solved by correctly configuring the network.

The objective of this project is to understand the fundamentals of TCP/IP addressing, subnetting, and subnet masks.

## Instructions

To run the project, download the folder available on the 42 Intra. Once downloaded, navigate to the folder and execute:

```bash
./run.sh
```

Your web browser should automatically open a page hosted on your localhost.

You should see something like this:

![alt text](https://github.com/Ryota49/img_netpractice/blob/main/screen_netpractice.png?raw=true)

There are two sections:

* **Training**: This section allows you to complete all exercises without a timer, helping you understand the different networking concepts and problems. You will also need to enter your Intra login in the field provided.
* **Evaluation**: During an evaluation at 42, you must complete 3 random levels selected from levels 6 to 10 of the Training section within 15 minutes. Once you are ready, click the **Start** button.

Everything available in this repository corresponds to my solutions for the exercises and is stored as `.json` files. The NetPractice application itself does not need to be submitted during the evaluation.
